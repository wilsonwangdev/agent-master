# Skills

Agent-usable skill definitions for this project.

## Quality Gate

A skill can only be added here when it meets ALL of the following:

1. Has been tested and verified in at least one mainstream agent environment (Claude Code, Cursor, etc.)
2. Follows the established skill format of the target agent tool
3. Solves a real, recurring need in this project's workflow
4. Does not duplicate functionality available through standard tool capabilities

## Format

Each skill is a single Markdown file describing: purpose, trigger conditions, input, steps, and expected output.
